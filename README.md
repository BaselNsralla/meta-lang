# Meta-lang



## Dependencies
The boost library, Because I use a regex thing from there. <br>

Download the boost library zip from somewhere <br>
put it in `/usr/local/include/` or someting


## Compile
You need gcc 9 or higher

### By hand
```bash
g++-9 -fconcepts -std=c++2a *.cpp
```

### With CMAKE
```bash
cmake ../build

```
